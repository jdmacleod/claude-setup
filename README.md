# claude-setup

A public reference repo for configuring and getting the most out of [Claude](https://claude.ai) and [Claude Code](https://claude.ai/code) — Anthropic's AI assistant and CLI tool.

## What's here

| Path | Contents |
|------|----------|
| `CLAUDE.md` | How Claude behaves in this repo + what CLAUDE.md files are and how to use them |
| `docs/tips-beginner.md` | Ten tips for new Claude Code users |

## Using this repo

Browse the docs directly on GitHub, or clone it to use the CLAUDE.md as a starting point for your own projects:

```bash
git clone https://github.com/jdmacleod/claude-setup.git
```

Copy and adapt `CLAUDE.md` into any project — Claude loads it automatically at the start of every session in that directory.
